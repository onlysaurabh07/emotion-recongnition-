# Emotion Recognition using FER Dataset

## Overview
This project is a facial emotion recognition system built using the FER (Facial Expression Recognition) dataset. It uses deep learning and computer vision techniques to detect and classify human emotions from facial expressions.

## Features
- Face detection from images or webcam
- Emotion classification using the FER dataset
- Real-time emotion prediction
- Supports multiple facial expressions

## Dataset
The model is trained on the **FER (Facial Expression Recognition)** dataset, which contains grayscale facial images categorized into emotions such as:
- Angry
- Disgust
- Fear
- Happy
- Sad
- Surprise
- Neutral

## Technologies Used
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Matplotlib

## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/emotion-recognition.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

## Results
The model predicts facial emotions using the trained FER model and displays the detected emotion for each detected face.
